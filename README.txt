CanopyApp - Analyze the forest's canopy with ease using technology you use every day -- your phone! Using your phone's gyroscope, each picture taken will be level with the ground and ensure accurate readings. You select the leaf colors with simple touches, so canopy data can be taken in any climate, any season! If you're looking for a densiometer or canopy analysis tool, this is a fantastic app for you!

This app is currently available at: https://apps.apple.com/us/app/canopyapp/id926943048
